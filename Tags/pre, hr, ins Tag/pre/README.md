# pre 태그란?
```
<pre> 요소 내의 텍스트는 시스템에서 미리 지정된 고정폭 글꼴을 사용하여 표현되며, 
텍스트에 사용된 여백과 줄바꿈이 모두 그대로 브라우저 화면에 나타내준다.
이러한 <pre> 요소를 사용하면, 독특한 서식의 텍스트나 컴퓨터 코드 등을 HTML 문서에 그대로 표현할 수 있습니다.
원래는 <br> 태그를 사용해서 줄바꿈을 해야했지만 <pre> 태그를 사용하면 예시처럼 사용할 수 있다.
```
## 예시 코드
***
```html
<pre>
        심일 연구소        제  1원     칙
        
        절대 삼일연애 하지          않을것  둘2째
        
        사일연애        도  하지        않을것
</pre>
```
