# dl & dt & dd
***
<dl> 요소의 요소들은 <table>의 요소들과 비슷하다고 생각해보면 편하다.
<dl> 요소는 설명 목록을 나타낸다. <dl>은 <dt>로 표기한 용어와 설명 그룹의 목록을 감싸서 설명 목록을 생성합니다. 
<dt>는 이제 테이블로 치면 th같은 요소로써 문단의 굵은 글씨처럼 표기되고 <dd> 태그는 그 아래 설명이 적힌다고 생각하면 된다.

```
    <caption>웹 언어</caption>
    <dl>
        <dt>HTML</dt>
        <dd>뼈대</dd>

        <dt>CSS</dt>
        <dd>성형</dd>

        <dt>JS</dt>
        <dd>장기이식</dd>
    </dl>
```
